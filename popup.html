This extension adds HSAC BrowserTest FitNesse Export to the Katalon Recorder extension. You must have it installed in order to use this extension.

Note: If you do not see BrowserTest as an option in the Export tab, please close Katalon Recorder and reopen it.

More Info on BrowserTest: (HSAC FitNesse Fixtures of GitHub)

This Extension is based on the Puppeteer Export extension by Sam Kirkland (GitHub)
